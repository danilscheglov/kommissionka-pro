# Komissionka PRO

> Лендинг комиссионного магазина `Комиссионка PRO` на `Vite`, `Tailwind CSS` и `vanilla JavaScript`.

## Стек

- `Vite`
- `Tailwind CSS`
- `JavaScript`

## Что нужно установить

Для локального запуска нужны:

- `Node.js 18+`
- `npm`

Проверка:

```bash
node -v
npm -v
```

## Запуск локально

Установить зависимости:

```bash
npm install
```

Запустить dev-сервер:

```bash
npm run dev
```

Обычно проект открывается по адресу:

```bash
http://localhost:5173
```

## Полезные команды

Разработка:

```bash
npm run dev
```

Сборка:

```bash
npm run build
```

Предпросмотр сборки:

```bash
npm run preview
```
